# BI Analyst Assessment - Aculocity

## Overview
This project is designed to conduct a comprehensive analysis as part of the BI Analyst assessment for Aculocity. The goal is to leverage data analytics to derive insights and support decision-making processes.

## Project Structure
The project is organized into several directories and files, each serving a specific purpose:

- **README.md**: This file provides an overview of the project.
- **docs/**: Contains documentation related to the assessment.
  - **assessment_instructions.md**: Instructions for completing the assessment.
  - **methodology_notes.md**: Notes on the methodologies used during the analysis.
  - **key_findings.md**: Summary of key findings from the assessment.
- **data/**: Directory for storing raw data files used in the assessment.
  - **raw/**: Contains raw data files in various formats.
- **sql/**: Contains SQL scripts for database operations.
  - **01_create_table.sql**: Script to create necessary tables.
  - **02_import_notes.md**: Notes on importing data into the database.
  - **03_customer_summary_view.sql**: Script to create a customer summary view.
  - **04_validation_checks.sql**: Script for data validation checks.
  - **05_top10_logic.sql**: Script for identifying top 10 entries based on criteria.
- **powerbi/**: Contains Power BI reports and visualizations.
  - **BI_Analyst_Assessment_Aculocity.pbix**: Power BI report file.
  - **dashboard_page1.png**: Screenshot of the first page of the Power BI dashboard.
- **outputs/**: Directory for output files.
  - **final_summary.pdf**: Final summary document of the assessment.

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Ensure you have the necessary software installed (e.g., SQL database, Power BI).
4. Follow the instructions in the `docs/assessment_instructions.md` file to complete the assessment.
