# Methodology Notes

This document outlines the methodologies employed during the BI Analyst assessment for Aculocity. The following approaches were utilized to ensure a comprehensive analysis:

## Data Collection
- **Source Identification**: Identified relevant data sources that align with the assessment objectives.
- **Data Extraction**: Extracted raw data from various sources, ensuring to maintain data integrity and accuracy.

## Data Cleaning
- **Preprocessing**: Conducted data cleaning to handle missing values, duplicates, and inconsistencies.
- **Transformation**: Applied necessary transformations to convert raw data into a suitable format for analysis.

## Analytical Techniques
- **Descriptive Analysis**: Utilized descriptive statistics to summarize the main characteristics of the data.
- **Exploratory Data Analysis (EDA)**: Conducted EDA to uncover patterns, trends, and insights within the data.
- **SQL Queries**: Developed SQL scripts to create tables, views, and perform validation checks to ensure data quality.

## Visualization
- **Power BI**: Leveraged Power BI to create interactive dashboards and visualizations that effectively communicate the findings.
- **Dashboard Design**: Focused on user-friendly design principles to enhance the interpretability of the visualizations.

## Reporting
- **Documentation**: Maintained thorough documentation of methodologies and findings to support transparency and reproducibility.
- **Key Findings**: Summarized key insights and conclusions drawn from the analysis, providing actionable recommendations.

This methodology framework ensures a structured approach to data analysis, facilitating informed decision-making based on the assessment outcomes.
